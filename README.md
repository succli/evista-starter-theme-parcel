# Evista Starter Wordpress Theme with Parcel

## Install

1. Run in theme root: `npm install`
2. Run `npm run build`

## Watch

`npm run watch`

## build

`npm run build`

### Notes

* Current version of Parcel Bundler work only with Node 8 and above.
